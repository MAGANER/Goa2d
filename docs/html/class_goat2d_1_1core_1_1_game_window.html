Detailed Description

GameWindow is class that is used to process game scene.

Author
MAGANER
Date
10.09.2023

Being derived from Scene manager it can add,change,process,draw .e.t.c each game scene. After the creation you should set scenes and check is everything ok to be sure the program won't crush.
