Detailed Description

Wrapper class for colored/outlined primitive - rectangle.

Author
Maganer
Version
0.1
Date
02.09.2023

This class is based on DrawableObject, but almost everything in core is just overwrapper for some low-level SDL features. So Rectangle is convenient class to draw/move/change color of low-level primitive.
